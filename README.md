# Me — Personal Dashboard

A personal dashboard and body-region explorer: interactive 3D model, stat panels, charts, and a live "Mind Stream" feed.

## Tech

- Vanilla JS, [Three.js](https://threejs.org/), [Chart.js](https://www.chartjs.org/), [Leaflet](https://leafletjs.com/)
- Static assets (HTML, CSS, JS) — no build step

## Run locally

Open `index.html` in a browser or serve the folder (e.g. `python -m http.server 8000`).

## Deploy to GitHub Pages

1. Push this repo to your GitHub account.
2. **Settings** → **Pages** → set source to **main** (root).
3. The site will be at `https://<username>.github.io/<repo-name>/`.
